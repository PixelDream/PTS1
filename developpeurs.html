C’est quoi une API ?

En informatique, une interface de programmation applicative (souvent désignée par le terme API pour Application Programming Interface) est un ensemble normalisé de classes, de méthodes ou de fonctions qui sert de façade par laquelle un logiciel offre des services à d'autres logiciels. Nous pouvons retrouvez l’API de Youtube qui met à disposition aux développeurs des accès aux services. Ainsi les développeurs peuvent intégrer des fonctionnalités Youtube dans leurs propres produits.

Comment ça marche ?

Android :

Android met à disposition des outils d'accessibilités au sein de son API. En effet, par le biais d’attribut dans les fichiers XML des Layouts il est possible de rendre accessible les éléments tels que les boutons, les inputs, les tableaux …

Exemple :

Dans ce fichier xml, on a ajouté un attribut à l’élément ImageView, l’attribut android:contentDescription=”...” qui permet de décrire le contenu de l’élément.

<ImageView ... android:contentDescription="@string/inspect" />
