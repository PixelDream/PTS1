La reconnaissance d'objets en quelques mots

La reconnaissance d’objets est une technique de Vision Artificiel (computer vision). Elle permet la reconnaisssance d'objet du quotidien dans une vidéo ou une photo. La reconnaissance d’objets fonctionne sur le principe d’algorithmes de Deep Learning et de Machine Learning. L'objectif de ces algorithmes est de reproduire le fonctionnement des yeux des humains et donc de reconnaitre les objets, les individus, les couleurs etc.

Pour aller plus loin : mathworks.com, blog.deloitte.fr

Exemple d'application utilisant la reconnaissance d'objet

LOOKOUT

Lookout est une application qui utilise la caméra des smartphones pour aider les personnes aveugles ou malvoyantes. Cette application comporte cinq modes :

  • mode lecture rapide ( texte lu instantanément à haute voix )
  • mode devise qui identifie les billets de banque américains
  • mode exploration : voix qui décrit tout ce qui peut être observé avec la caméra
  • mode analyse de document (photo ou capture d’écran)
  • mode code barre pour identifier un produit
